# Vite

## 비트란
- Create React App 대신에 요즘 많이 사용하는 차세대 번들러
- cf) 발음은 '비트'
- cf) CRA는 22년 4월 이후 업데이트 X

## 강점
1. 간편하게 리액트 프로젝트 생성 가능
2. 빠르고 쾌적한 개발 서버 사용 가능

## vite 프로젝트 시작하기
```
npm create vite@latest
```

- React, TypeScript 등 사용할 패키지 선택하고 생성

```
npm install vite
```

로컬에서 프로그램 실행하기
```
npm run dev
```
